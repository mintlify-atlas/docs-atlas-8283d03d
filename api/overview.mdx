The TravelConnect API is a REST API that lets you manage trips, users, chat messages, and admin operations.

**Base URL:** `http://localhost:8000`

## Authentication

Most endpoints require a Bearer JWT token in the `Authorization` header. Obtain a token by calling `POST /auth/login`.

```http
Authorization: Bearer <your_token>
```

<CodeGroup>

```bash cURL
curl http://localhost:8000/viatge/trips \
  -H "Authorization: Bearer <your_token>"
```

```javascript axios
import axios from 'axios';

const response = await axios.get('http://localhost:8000/viatge/trips', {
  headers: {
    Authorization: `Bearer ${token}`,
  },
});
```

</CodeGroup>

## HTTP status codes

| Code | Meaning |
|------|---------|
| `200` | Request succeeded |
| `400` | Bad request — invalid input or business rule violation |
| `401` | Unauthorized — missing or invalid token |
| `403` | Forbidden — insufficient role permissions |
| `404` | Not found — resource does not exist |
