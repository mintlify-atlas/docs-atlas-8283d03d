## What is TravelConnect?

TravelConnect is a web application that brings travelers together. It lets users discover group trips, join adventures, and communicate with fellow travelers — all in one place.

The platform has three distinct user roles, each with different capabilities:

- **Viatger (Traveler)** — Browse available trips, enroll in trips, and chat with other participants
- **Creador (Trip Creator)** — Everything a Traveler can do, plus create and manage trips
- **Admin** — Full platform access including user management and promotion approvals

## Tech stack

TravelConnect is built with a modern, decoupled architecture:

| Layer | Technology |
|---|---|
| Frontend | React 19, Vite, React Router v7, Bootstrap 5 |
| Backend | Python, FastAPI, SQLAlchemy ORM |
| Auth | JWT (JSON Web Tokens) with role claims |
| HTTP Client | Axios |

## Getting started

<Steps>
  <Step title="Set up the backend">
    Install Python dependencies and start the FastAPI server. See the [backend setup guide](/guides/backend-setup).
  </Step>
  <Step title="Set up the frontend">
    Install Node.js dependencies and start the Vite dev server. See the [frontend setup guide](/guides/frontend-setup).
  </Step>
  <Step title="Register an account">
    Visit `http://localhost:5173/register` to create your first account. New users start with the **Viatger** role.
  </Step>
  <Step title="Explore trips">
    Log in and navigate to the dashboard to see available trips. Enroll in one to join the trip chat.
  </Step>
</Steps>
